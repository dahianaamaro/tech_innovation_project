# The Effects of Population Size on Technological Innovation

## Overview
This project analyzes the relationship between population size and technological innovation using World Bank data from 1959–2000.

Innovation is measured using patent data, while population represents total population across countries worldwide.

## Tools Used
- Python
- pandas
- numpy
- scipy
- seaborn
- matplotlib
- statsmodels

## Methods
- Data cleaning
- Correlation analysis
- Log regression analysis
- Data visualization

## Findings
The analysis found a positive relationship between population size and patent activity. Regression results showed that increases in population were associated with increases in innovation.

## Dataset
World Bank Database:
“Innovation and Development Around the World”
